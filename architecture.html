biwenger-intel-proxy.html
App principal. React 18 + Babel CDN. Tabs: Saldos, Tablón, Mercado, Jugadores, Stats. localStorage cache. SERVER_URL auto-detect (localhost vs GitHub Pages).
React SPA · frontend
server.js
Servidor Express. Proxy /api/* → biwenger.as.com. Proxy /cfapi/* → cf.biwenger.com. API local /api-local/*. Cron de alertas cada 5 min. nodemailer.
