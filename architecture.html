server.js
Servidor Express. Proxy hacia biwenger.as.com y cf.biwenger.com. API local para token y alertas. Cron de mercado con nodemailer.

App principal. React 18 + Babel CDN. Tabs: Saldos, Tablón, Mercado, Jugadores, Stats. Caché en localStorage. Se sirve desde el proxy.
